## 原子任务列表

### 任务1：项目深度分析
- **输入契约**：完整项目源码、文档、示例
- **输出契约**：项目结构分析报告
- **实现约束**：不修改源码，仅分析
- **验收标准**：准确识别所有模块和依赖关系

### 任务2：架构总览文档
- **输入契约**：项目分析结果
